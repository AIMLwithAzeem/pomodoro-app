# Calm & Modern Pomodoro Timer

A simple, aesthetic Pomodoro timer to help you stay focused.

## Features
- **Default Timer**: Starts at 25 minutes.
- **Custom Timer**: Enter any duration (1-120 minutes).
- **Controls**: Start, Pause, and Reset functionality.
- **Audio Alert**: Plays a gentle chime when the timer ends.
- **Visuals**: Modern dark theme with a progress ring.

## How to Run
1. Navigate to the folder where you saved the files (likely `Desktop/Antigravity-Projects/pomodoro-app`).
2. Double-click `index.html` to open it in your default web browser.

## Tech Stack
- HTML5
- CSS3 (Flexbox, CSS Variables, Animations)
- JavaScript (Vanilla)
